# Support

这个仓库适合以下支持请求：

- 文档错误或缺失
- 案例说明不清晰
- 样例 SQL 明显有误
- 高频业务场景建议
- 本地最小运行步骤问题

## 提问时请附带

- Flink 版本
- 运行方式
- 执行的 SQL 文件或最小 SQL
- 完整报错信息
- 复现步骤

## 不在支持范围内

- 一对一环境排障
- 私有业务 SQL 代写或改写
- 长篇教学辅导
- 与仓库无关的大数据技术咨询

## 建议入口

- 可复现问题：提 Bug Issue
- 新案例建议：提 Feature Request
- 小型修正：直接发 PR

Issue 模板位于 [`.github/ISSUE_TEMPLATE`](./.github/ISSUE_TEMPLATE)。
