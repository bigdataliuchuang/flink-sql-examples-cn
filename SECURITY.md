# Security Policy

这是一个以学习和案例演示为主的仓库，不提供生产级安全承诺。

## 如何报告安全问题

如果你发现：

- 仓库中误提交了敏感信息
- 文档中包含不应公开的访问方式或凭据
- 示例内容可能引导出明显危险的错误使用方式

请不要通过公开 Issue 披露细节。

请直接发送邮件到：

- `lczxy1996@163.com`

邮件标题建议包含 `flink-sql-examples-cn security`，便于识别。

## 范围说明

- 这个仓库主要包含 SQL 样例、文档和轻量本地运行说明
- 不提供生产环境安全基线、漏洞响应 SLA 或长期安全维护承诺
