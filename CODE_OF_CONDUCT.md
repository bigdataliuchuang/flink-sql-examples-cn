# Code of Conduct

## Our Standard

为保证这个仓库的讨论保持高效和可协作，请尽量做到：

- 尊重他人，聚焦问题本身
- 使用清晰、可复现、可验证的描述
- 接受不同经验水平的参与者
- 避免人身攻击、羞辱、骚扰或恶意引战

## Unacceptable Behavior

以下行为不被接受：

- 侮辱、歧视、骚扰
- 与技术问题无关的攻击性言论
- 恶意刷屏或反复提交低质量内容
- 公开披露他人的敏感信息

## Scope

本规范适用于仓库的 Issue、PR 和相关公开讨论。

## Enforcement

维护者可以删除明显不合适的内容，并在必要时限制互动。

如有严重问题，请通过 [SECURITY.md](./SECURITY.md) 中的联系方式私下反馈。
